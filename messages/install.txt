Deviot
-------------------------------
Sublime Text plugin for IoT development

** IMPORTANT! **

This plugin require Python 2, if after install python you are experienced any problems go to Deviot Menu > Options > Remove Preferences File, and restart Sublime Text

If after that you continue having problem, please submit an issue explaining your problem from this link: https://github.com/gepd/Deviot/issues/new

--------------------------------

** ¡IMPORTANTE! **
    
Este plugin requiere Python 2, si después de instalar python experimentas algún problema, anda a Menu Deviot > Opciones > Eliminar Archivos de Prefencias, y reinicia Sublime Text.
